Modelo Osi

O modelo “Open System Interconnection” é um modelo de camadas de rede criado em 1984 pela ISO (“International Organization for Standardization”) para proporcionar a interconexão de sistemas abertos, visando a inoperabilidade do sistema em questão.

No modelo OSI os protocolos são agrupados em 7 camadas, iniciando sempre pela camada inferior.

TCP/IP

O Modelo TCP/IP é um conjunto de protocolos de comunicação entre hosts distintos, que garante a qualidade dos dados transmitidos e fornecendo uma boa comunicação entre transmissor e receptor.
Este modelo é composto por 4 camadas independentes que garantem a integridade e a entrega adequada dos dados trafegados do que será executado na camada superior.

Acesso à Rede

Primeira camada do modelo TCP/IP é responsável pela comunicação entre a rede física e as camadas superiores, definindo especificações para transmissão de bits sobre o meio físico por meio do encapsulamento dos dados transmitidos e do endereçamento MAC. Principais protocolos envolvidos: Ethernet, ARP, Wi-fi, entre outros.

Internet ou Inter-Rede

Realiza o roteamento dos pacotes de dados - transmissão de sinais elétricos - entre redes distintas e gerencia endereços IP, determinando a melhor rota para a entrega dos dados. Principais protocolos envolvidos: IP, ARP, RIP, entre outros.

Transporte

Garante a entrega correta e ordenada dos dados entre os diferentes hosts envolvidos, por meio dos protocolos UDP – controle de fluxo de informações - e TCP – entrega rápida e eficiente dos dados. 

Aplicação

Por conter todos os protocolos de nível mais alto, esta última camada executa a interação humano-máquina, fornecendo a interface do software para aplicativos e serviços utilizados pelos usuários finais. Principais protocolos envolvidos: HTTP(S), FTP, DNS, entre outros.
